Xebia LMS Technical Documentation

Architecture, frontend, backend, APIs, features, execution flows, database, and deployment guidance for the current Admin, Teacher, and Student portals.

Important: Admin and Assessment data can use the Spring API when enabled. Batch workspaces and offline Assessment mode use browser localStorage. The backend uses tenant and mock user headers, not JWT authentication.
